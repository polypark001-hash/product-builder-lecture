# Lotto Number Recommender

A simple and modern Lotto Number Recommendation site built with vanilla HTML, CSS, and JavaScript.

## Features
- **Random Number Generation:** Generates 6 unique numbers (1-45).
- **Modern UI:** Clean, responsive design with smooth animations.
- **Dark/Light Mode:** Toggle between themes with automatic persistence via `localStorage`.
- **Responsive:** Optimized for both desktop and mobile devices.

## How to Use
1. Click the **"Generate Numbers"** button to get your lucky lotto numbers.
2. Use the **"Dark Mode" / "Light Mode"** button at the top to switch themes.
3. Your theme preference will be saved for your next visit.

## Deployment
This project is deployed to the Git repository and is ready for static hosting.
