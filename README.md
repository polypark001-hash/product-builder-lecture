# Dinner Menu Recommender

A stylish and modern web application that helps you decide what's for dinner! Transform your indecision into a delicious meal recommendation with a single click.

## Features
- **Diverse Menu Recommendations:** Picks from a curated list of global cuisines (Korean, Italian, Japanese, Mexican, and more).
- **Modern & Appetizing UI:** Clean, warm design optimized for a great user experience.
- **Theme Support:** Supports both **Light** and **Dark** modes, automatically saving your preference.
- **Animated Experience:** Smooth animations to make the recommendation process feel interactive.
- **Responsive Design:** Works beautifully on mobile, tablet, and desktop screens.

## How to Use
1. Open the site in your browser.
2. Click the **"Get Recommendation"** button.
3. Wait for the animation to reveal your dinner for tonight!
4. Use the **"Dark Mode" / "Light Mode"** toggle at the top to change the theme.

## Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Modern styling with CSS variables, Flexbox, and keyframe animations.
- **JavaScript (ES6+):** Dynamic content generation and theme persistence via `localStorage`.
- **Git:** Version control and deployment.
